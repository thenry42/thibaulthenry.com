Promptly

GitHub
Dart Flutter OpenAI API Anthropic API Gemini API Mistral API DeepSeek API Ollama Python

A Flutter application that allows you to interact with AI models. It uses multiple AI/LLM providers APIs to generate text and the Ollama API to run local models. It's my first Flutter app and I'm still learning the basics. The app can safely save API keys and chat history between sessions. It is nowhere near production ready but I intend on releasing a stable version in the near future for Linux, MacOS and Windows. Frontend is written in Dart and Flutter, backend is written in Python (FastAPI).

ft_irc

Subject
C++ Sockets IRC Protocol

ft_irc is a 3-person project that consist of re-creating an IRC chat in C++ (only). We had to manage connections, logging-in, logging-out, adding users to channels, blocking users, etc. We used IRSSI as a client and it worked flawlessly.

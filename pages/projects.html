My Terminal Portfolio

GitHub
HTML CSS JavaScript Hostinger

A terminal-style portfolio website built with vanilla JavaScript. Features a fully interactive command-line interface with content history navigation and hack effect text rendering. This project is a work in progress and I'm currently working on adding more features to it. The main goal of this website is to showcase my skills and projects in a interactive way.

Promptly

GitHub
Dart Flutter OpenAI API Anthropic API Ollama

A Flutter application that allows you to interact with AI models. It uses the OpenAI and Anthropic APIs to generate text and the Ollama API to run local models. It's my first Flutter app and I'm still learning the basics. The app can safely save API keys and chat history between sessions. It is nowhere near production ready but I intend on releasing a stable version in the near future for Linux, MacOS and Windows.
